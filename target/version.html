シグニチャ

m.version

引数 型 必須 説明
returns String バージョン番号を返します

どのように動作するのか

m.versionは現在のリリースのsemver文字列が入ったプロパティです。

Semver（またはセマンティックバージョニング）ではバージョン番号は"0.0.0"という文法に従わなければなりません。最初の数値がメジャー、次がマイナー、３番目がパッチ番号です。

  • MAJOR番号が変わるときはAPIの後方互換性のない変更が含まれます。
  • MINOR番号が変わる時は後方互換性を守った中で機能が追加されます。
  • PATCH番号が変わる時は、後方互換性がある状態でバグ修正が行われます。
